什么是导数？

‌导数（Derivative）‌ 是微积分中的核心概念，它描述了一个函数在某一点处的 ‌瞬时变化率‌。通俗来说：

  • 导数就是函数在某点的 ‌变化速度‌
  • 几何意义：函数曲线在该点的 ‌切线斜率‌
  • 物理意义：位移函数的导数是速度，速度函数的导数是加速度

‌数学定义‌：
函数 y=f(x) 在点 x0 处的导数定义为：

f′(x0)=lim⁡Δx→0 f(x0+Δx)−f(x0)Δxf′(x0)=Δx→0limΔx**f(x0+Δx)−f(x0)

如何求导？（求导方法详解）

一、基本函数求导公式（必须掌握）

常数函数 c 0
幂函数 xn nxn−1
指数函数 ex ex
指数函数 ax axln⁡a
对数函数 ln⁡x 1x
三角函数 sin⁡x cos⁡x
cos⁡x −sin⁡x
tan⁡x sec⁡2x

二、求导法则（组合函数求导）

  1. ‌加法法则‌：

    (f+g)′=f′+g′(f+g)′=f′+g′

    例： (x2+sin⁡x)′=2x+cos⁡x(x2+sinx)′=2x+cosx

  2. ‌乘法法则‌：

    (f⋅g)′=f′g+fg′(f⋅g)′=f′g+f**g′

    例： (x⋅ex)′=1⋅ex+x⋅ex=ex(1+x)(x⋅e**x)′=1⋅e**x+x⋅e**x=e**x(1+x)

  3. ‌除法法则‌：

    (fg)′=f′g−fg′g2(g**f)′=g2f′g−f**g′

    例： (sin⁡xx)′=xcos⁡x−sin⁡xx2(xsinx)′=x2xcosx−sinx

  4. ‌链式法则‌（复合函数求导）：

    ddxf(g(x))=f′(g(x))⋅g′(x)dxd**f(g(x))=f′(g(x))⋅g′(x)

    例： (sin⁡(2x))′=cos⁡(2x)⋅2=2cos⁡(2x)(sin(2x))′=cos(2x)⋅2=2cos(2x)

三、特殊求导技巧

  1. ‌隐函数求导‌：
    当函数以方程形式给出（如 x2+y2=1x2+y2=1）

    • 两边同时对 xx 求导
    • 解出 dydxdxd**y

    x2+y2=12x+2ydydx=0dydx=−xyx2+y22x+2ydxdydxdy=1=0=−y**x

  2. ‌对数求导法‌（适用于复杂函数）：

    • 取自然对数：ln⁡y=ln⁡f(x)lny=lnf(x)
    • 两边求导
    • 解出 y′y′

    例：y=xxln⁡y=xln⁡x1yy′=ln⁡x+x⋅1xy′=xx(ln⁡x+1)例：ylny**y1y′y′=x**x=xlnx=lnx+x⋅x1=x**x(lnx+1)

四、求导步骤总结（实战流程）

  1. ‌识别函数类型‌（基本函数/组合函数）
  2. ‌选择求导法则‌（公式/四则运算/链式法则）
  3. ‌逐层求导‌（复合函数从外向内）
  4. ‌化简表达式‌（合并同类项等）

导数应用场景

  1. 物理学：速度/加速度计算
  2. 经济学：边际成本/收益分析
  3. 工程学：系统变化率优化
  4. 生物学：种群增长率研究

📌 ‌关键记忆点‌：

  • 导数 = 瞬时变化率 = 切线斜率
  • 幂函数求导：指数前置，指数减 1
  • 复合函数求导：链式法则（从外向内）
  • 遇到复杂函数：尝试取对数再求导
