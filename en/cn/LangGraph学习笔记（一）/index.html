йҡҸзқҖAIеӨ§жЁЎеһӢеҸ‘еұ•пјҢе°Өе…¶з”ҹжҲҗејҸAIеӨ§жЁЎеһӢзҡ„еҸ‘еұ•пјҢе…¶еңЁеә”з”ЁйўҶеҹҹд№ҹеҫ—д»ҘдәҶе№ҝжіӣзҡ„еә”з”ЁпјҢеҰӮеңЁдҝЎиҙ·йўҶеҹҹпјҢдәәд»¬жҠҠеӨ§жЁЎеһӢеә”з”ЁеҲ°дҝЎиҙ·зҡ„еҗ„дёӘзҺҜиҠӮпјҢеҰӮеӮ¬ж”¶пјҢйҖҡиҝҮAgentдёҺAIеӨ§жЁЎеһӢз»“еҗҲпјҢи®©AIжқҘзҗҶи§Је’ҢеҲҶжһҗз”ЁжҲ·зҡ„иҝҳж¬ҫж„ҸеӣҫпјҢе°Ҷз»ҷеҮәеҗҲзҗҶзҡ„иҜқжңҜпјҢжңҖеҗҺйҖҡиҝҮз”ҹжҲҗзҡ„AIиҜӯйҹідёҺз”ЁжҲ·иҝӣиЎҢиҜӯйҹіжІҹйҖҡпјҢжһҒеӨ§зҡ„еҮҸе°‘дәҶдәәе·ҘеӮ¬ж”¶зҡ„жҲҗжң¬гҖӮеҸҲеҰӮеңЁиҗҘй”ҖйўҶеҹҹпјҢејҖеҸ‘иҖ…д»¬е°ҶпјҢиҗҘй”Җдёӯзҡ„еҗ„дёӘзҺҜиҠӮе°ҒиЈ…жҲҗAgentпјҢйҖҡиҝҮWorkFlowе°ҶAgentж•ҙеҗҲиө·жқҘпјҢжҜҸдёӘAgentдёҺеӨ§жЁЎеһӢиҝӣиЎҢдәӨдә’пјҢе№¶йҖҡиҝҮи°ғз”Ёе·Ҙе…·е®ҢжҲҗиҜҘAgentзҡ„д»»еҠЎпјҢжңҖеҗҺе°ҶжүҖжңүзҡ„е·ҘдҪңеҶ…е®№еҒҡдёәж•ҙдҪ“иҝӣиЎҢиҫ“еҮәеҲ¶е®ҡе®Ңж•ҙзҡ„иҗҘй”Җи®ЎеҲ’пјҢеҗҢж—¶еҜ№иҗҘй”Җзҡ„з»“жһңйҖҡиҝҮж•°жҚ®еҲҶжһҗзҡ„AgentиҝӣиЎҢж•°жҚ®еӣһеҪ’еҲҶжһҗпјҢз»ҷиҗҘй”Җдәәе‘ҳжҸҗдҫӣзӣёе…ізҡ„ж•°жҚ®еҲҶжһҗе’ҢиҗҘй”Җзӯ–з•Ҙзҡ„ж”Ҝж’‘гҖӮ

еңЁдәҶи§ЈLangGraphжҳҜпјҢжҲ‘д»¬е…ҲиҰҒдәҶи§ЈдёҖдёӢеңЁAIйўҶеҹҹдёӯзҡ„дёҖдәӣеҹәжң¬жҰӮеҝөпјҢеҰӮAgentгҖҒеӨ§иҜӯиЁҖжЁЎеһӢгҖҒMCPгҖҒе·Ҙе…·гҖҒFunction Callзӯү

дёҖгҖҒд»Җд№ҲжҳҜеӨ§иҜӯиЁҖжЁЎеһӢ(LLM)

еӨ§еһӢиҜӯиЁҖжЁЎеһӢпјҲLLMпјүжҳҜејәеӨ§зҡ„AIе·Ҙе…·пјҢиғҪеӨҹеғҸдәәзұ»дёҖж ·и§ЈйҮҠе’Ңз”ҹжҲҗж–Үжң¬гҖӮе®ғд»¬еҠҹиғҪеӨҡж ·пјҢеҸҜд»Ҙж’°еҶҷеҶ…е®№гҖҒзҝ»иҜ‘иҜӯиЁҖгҖҒжҖ»з»“е’Ңеӣһзӯ”й—®йўҳпјҢиҖҢж— йңҖй’ҲеҜ№жҜҸдёӘд»»еҠЎиҝӣиЎҢдё“й—Ёи®ӯз»ғгҖӮ

йҷӨдәҶж–Үжң¬з”ҹжҲҗпјҢи®ёеӨҡжЁЎеһӢиҝҳж”ҜжҢҒпјҡ

  • е·Ҙе…·и°ғз”Ё - и°ғз”ЁеӨ–йғЁе·Ҙе…·пјҲеҰӮж•°жҚ®еә“жҹҘиҜўжҲ–APIи°ғз”Ёпјүе№¶еңЁе…¶е“Қеә”дёӯдҪҝз”Ёз»“жһңгҖӮ
  • з»“жһ„еҢ–иҫ“еҮә - жЁЎеһӢе“Қеә”иў«йҷҗеҲ¶дёәйҒөеҫӘе®ҡд№үзҡ„ж јејҸгҖӮ
  • еӨҡжЁЎжҖҒ - еӨ„зҗҶе’Ңиҝ”еӣһйҷӨж–Үжң¬д№ӢеӨ–зҡ„ж•°жҚ®пјҢдҫӢеҰӮеӣҫеғҸгҖҒйҹійў‘е’Ңи§Ҷйў‘гҖӮ
  • жҺЁзҗҶ - жЁЎеһӢжү§иЎҢеӨҡжӯҘжҺЁзҗҶд»Ҙеҫ—еҮәз»“и®әгҖӮ

жЁЎеһӢжҳҜAgentзҡ„жҺЁзҗҶеј•ж“ҺпјҢе®ғд»¬й©ұеҠЁAgentзҡ„еҶізӯ–иҝҮзЁӢпјҢзЎ®е®ҡи°ғз”Ёе“Әдәӣе·Ҙе…·гҖҒеҰӮдҪ•и§ЈйҮҠз»“жһңд»ҘеҸҠдҪ•ж—¶жҸҗдҫӣжңҖз»Ҳзӯ”жЎҲ гҖӮ

дәҢгҖҒд»Җд№ҲжҳҜAgent

Agent еҚіжҷәиғҪдҪ“пјҢAgentе°ҶиҜӯиЁҖжЁЎеһӢдёҺе·Ҙе…·з»“еҗҲпјҢеҲӣе»әиғҪеӨҹеҜ№д»»еҠЎиҝӣиЎҢжҺЁзҗҶгҖҒеҶіе®ҡдҪҝз”Ёе“Әдәӣе·Ҙе…·е№¶иҝӯд»Јең°и§ЈеҶій—®йўҳзҡ„зі»з»ҹгҖӮдёҖдёӘLLMжҷәиғҪдҪ“еңЁеҫӘзҺҜдёӯиҝҗиЎҢе·Ҙе…·д»Ҙе®һзҺ°зӣ®ж ҮгҖӮжҷәиғҪдҪ“дёҖзӣҙеңЁиҝҗиЎҢзӣҙеҲ°ж»Ўи¶іеҒңжӯўжқЎд»¶ вҖ”вҖ”вҖ” еҚіжЁЎеһӢеҸ‘еҮәжңҖз»Ҳиҫ“еҮәжҲ–иҫҫеҲ°иҝӯд»ЈйҷҗеҲ¶гҖӮ

дёүгҖҒд»Җд№Ҳе·Ҙе…·

и®ёеӨҡAIеә”з”ЁзЁӢеәҸйҖҡиҝҮиҮӘз„¶иҜӯиЁҖдёҺз”ЁжҲ·дәӨдә’гҖӮз„¶иҖҢпјҢжҹҗдәӣз”ЁдҫӢиҰҒжұӮжЁЎеһӢдҪҝз”Ёз»“жһ„еҢ–иҫ“е…ҘзӣҙжҺҘдёҺеӨ–йғЁзі»з»ҹпјҲеҰӮAPIгҖҒж•°жҚ®еә“жҲ–ж–Үд»¶зі»з»ҹпјүиҝӣиЎҢжҺҘеҸЈдәӨдә’гҖӮ

е·Ҙе…·жҳҜжҷәиғҪдҪ“и°ғз”Ёд»Ҙжү§иЎҢж“ҚдҪңзҡ„з»„д»¶гҖӮе®ғд»¬йҖҡиҝҮи®©жЁЎеһӢйҖҡиҝҮе®ҡд№үжҳҺзЎ®зҡ„иҫ“е…Ҙе’Ңиҫ“еҮәдёҺдё–з•ҢдәӨдә’жқҘжү©еұ•жЁЎеһӢеҠҹиғҪгҖӮе·Ҙе…·е°ҒиЈ…дәҶдёҖдёӘеҸҜд»Ҙи°ғз”ЁеҮҪж•°еҸҠе…¶иҫ“е…ҘжЁЎејҸгҖӮиҝҷдәӣеҸҜд»ҘеҸҜд»Ҙдј йҖ’з»ҷе…је®№зҡ„иҒҠз”ҹжЁЎеһӢпјҢе…Ғи®ёжЁЎеһӢеҶіе®ҡжҳҜеҗҰи°ғз”Ёе·Ҙе…·д»ҘеҸҠдҪҝз”Ёдёӯе“ӘдәӣеҸӮж•°гҖӮеңЁиҝҷдәӣеңәжҷҜдёӯпјҢе·Ҙе…·и°ғз”ЁдҪҝеҫ—жЁЎеһӢиғҪеӨҹз”ҹжҲҗз¬ҰеҗҲжҢҮе®ҡиҫ“е…ҘжЁЎејҸзҡ„иҜ·жұӮгҖӮ

еӣӣгҖҒд»Җд№ҲжҳҜMCP

MCPпјҲModel Context ProtocolпјҢжЁЎеһӢдёҠдёӢж–ҮеҚҸи®®пјүжҳҜеӨ§еһӢиҜӯиЁҖжЁЎеһӢпјҲLLMпјүдёҺеӨ–йғЁе·Ҙе…·гҖҒж•°жҚ®жәҗиҝӣиЎҢдәӨдә’зҡ„йҖҡз”Ёж ҮеҮҶеҚҸи®®пјҢз”ұAnthropicе…¬еҸёеңЁ2024е№ҙ11жңҲжҺЁеҮәвҖҢ2гҖӮе®ғзұ»дјјдәҺAIйўҶеҹҹзҡ„вҖңUSB-CжҺҘеҸЈвҖқпјҢдёәжЁЎеһӢдёҺеӨ–йғЁиө„жәҗпјҲеҰӮж•°жҚ®еә“гҖҒжң¬ең°ж–Үд»¶зӯүпјүжҸҗдҫӣж ҮеҮҶеҢ–иҝһжҺҘж–№ејҸпјҢе®һзҺ°еҸҢеҗ‘йҖҡдҝЎвҖҢгҖӮ

дә”гҖҒд»Җд№ҲжҳҜFunction Call

Function CallingпјҲеҮҪж•°и°ғз”ЁпјүжҳҜеӨ§еһӢиҜӯиЁҖжЁЎеһӢпјҲLLMпјүзҡ„дёҖйЎ№е…ій”®жҠҖжңҜиғҪеҠӣпјҢе®ғдҪҝжЁЎеһӢиғҪеӨҹиҜҶеҲ«дҪ•ж—¶йңҖиҰҒи°ғз”ЁеӨ–йғЁеҮҪж•°пјҢе№¶з”ҹжҲҗз¬ҰеҗҲеҮҪж•°иҰҒжұӮзҡ„еҸӮж•°пјҢд»ҺиҖҢдёҺеӨ–йғЁе·Ҙе…·гҖҒAPIе’ҢжңҚеҠЎиҝӣиЎҢдәӨдә’вҖҢгҖӮиҝҷйЎ№жҠҖжңҜи®©AIжЁЎеһӢд»ҺвҖңеҸӘдјҡиҜҙиҜқвҖқеҸҳдёәвҖңиғҪеӨҹиЎҢеҠЁвҖқпјҢйҖҡиҝҮи°ғз”ЁеӨ–йғЁеҮҪж•°е®һзҺ°ж•°жҚ®жҹҘиҜўгҖҒи®Ўз®—гҖҒжҺ§еҲ¶еӨ–йғЁзі»з»ҹзӯүж— жі•йҖҡиҝҮзәҜиҜӯиЁҖз”ҹжҲҗе®ҢжҲҗзҡ„д»»еҠЎвҖҢгҖӮ

ж ёеҝғеҺҹзҗҶ

Function Callingзҡ„ж ёеҝғеҺҹзҗҶеҹәдәҺд»ҘдёӢе…ій”®жңәеҲ¶вҖҢпјҡ

  • вҖҢеҮҪж•°жҸҸиҝ°вҖҢпјҡеҗ‘жЁЎеһӢжҸҗдҫӣеҸҜз”ЁеҮҪж•°зҡ„иҜҰз»ҶжҸҸиҝ°пјҢеҢ…жӢ¬еҮҪж•°еҗҚз§°гҖҒеҸӮж•°зұ»еһӢгҖҒеҠҹиғҪиҜҙжҳҺзӯүгҖӮ
  • вҖҢз»“жһ„еҢ–иҫ“еҮәвҖҢпјҡжЁЎеһӢиғҪд»ҘJSONжҲ–е…¶д»–йў„е®ҡд№үж јејҸжҸҗдҫӣеҸӮж•°пјҢиҖҢдёҚжҳҜз”ҹжҲҗиҮӘз”ұж–Үжң¬вҖҢгҖӮ
  • вҖҢж„ҸеӣҫиҜҶеҲ«дёҺеҸӮж•°з”ҹжҲҗвҖҢпјҡжЁЎеһӢи§Јжһҗз”ЁжҲ·иҫ“е…ҘпјҲеҰӮвҖңжҹҘиҜўеҢ—дә¬еӨ©ж°”вҖқпјүпјҢеҲӨж–ӯжҳҜеҗҰйңҖиҰҒи°ғз”ЁеӨ–йғЁеҮҪж•°пјҢе№¶иҫ“еҮәз»“жһ„еҢ–жҢҮд»ӨпјҲеҰӮ{"name": "get_weather", "arguments": {"location": "еҢ—дә¬"}}пјүгҖӮ

дё»иҰҒз”ЁйҖ”

Function Callingзҡ„дё»иҰҒз”ЁйҖ”жҳҜи®©еӨ§иҜӯиЁҖжЁЎеһӢйҖҡиҝҮиҮӘз„¶иҜӯиЁҖз”ҹжҲҗAPIи°ғз”ЁеҸӮж•°пјҢй—ҙжҺҘи®ҝй—®еӨ–йғЁзі»з»ҹзҡ„ж•°жҚ®пјҢжҜ”еҰӮеӨ©ж°”гҖҒиӮЎзҘЁгҖҒж•°жҚ®еә“зӯүвҖҢ3гҖӮе®ғи§ЈеҶідәҶеӨ§жЁЎеһӢзҹҘиҜҶж— жі•е®һж—¶жӣҙж–°зҡ„й—®йўҳвҖҢ

е…ӯгҖҒMCPдёҺFunction Callingзҡ„еҢәеҲ«

MCPеҚҸи®®пјҲModel Context ProtocolпјүдёҺFunction CallingжҳҜеӨ§жЁЎеһӢдёҺеӨ–йғЁе·Ҙе…·дәӨдә’зҡ„дёӨз§ҚдёҚеҗҢжңәеҲ¶пјҢе®ғд»¬еңЁе®ҡдҪҚгҖҒжһ¶жһ„е’Ңеә”з”ЁеңәжҷҜдёҠеӯҳеңЁжҳҫи‘—е·®ејӮгҖӮд»ҘдёӢжҳҜе…·дҪ“еҜ№жҜ”пјҡ

е®ҡдҪҚдёҺзӣ®ж ҮвҖҢ

  • вҖҢMCPеҚҸи®®вҖҢпјҡз”ұAnthropicжҸҗеҮәзҡ„вҖҢејҖж”ҫж ҮеҮҶеҚҸи®®вҖҢпјҢж—ЁеңЁз»ҹдёҖдёҚеҗҢеӨ§жЁЎеһӢдёҺеӨ–йғЁе·Ҙе…·/ж•°жҚ®жәҗзҡ„дәӨдә’ж–№ејҸпјҢзұ»дјјвҖңAIз•Ңзҡ„USB-CжҺҘеҸЈвҖқвҖҢгҖӮе…¶зӣ®ж ҮжҳҜи§ЈеҶізі»з»ҹйӣҶжҲҗж ҮеҮҶеҢ–е’ҢдёҠдёӢж–Үдј иҫ“дјҳеҢ–й—®йўҳпјҢжһ„е»әAIеә”з”Ёзҡ„вҖңйҖҡз”Ёй«ҳйҖҹе…¬и·ҜвҖқвҖҢгҖӮ
  • вҖҢFunction CallingвҖҢпјҡз”ұOpenAIзӯүеҺӮе•ҶжҸҗдҫӣзҡ„вҖҢзү№е®ҡеҠҹиғҪвҖҢпјҢе…Ғи®ёжЁЎеһӢйҖҡиҝҮиҮӘз„¶иҜӯиЁҖжҢҮд»Өи°ғз”Ёйў„е®ҡд№үзҡ„еҮҪж•°пјҲеҰӮеӨ©ж°”жҹҘиҜўгҖҒAPIи°ғз”ЁпјүпјҢеұһдәҺвҖңе·Ҙе…·еһӢжҺҘеҸЈвҖқвҖҢгҖӮ

жһ¶жһ„и®ҫи®ЎвҖҢ

  • вҖҢMCPвҖҢпјҡйҮҮз”ЁвҖҢе®ўжҲ·з«Ҝ-жңҚеҠЎеҷЁпјҲC/Sпјүжһ¶жһ„вҖҢпјҢж”ҜжҢҒе®һж—¶еҸҢеҗ‘йҖҡдҝЎгҖҒе·Ҙе…·иҮӘеҠЁеҸ‘зҺ°е’Ңй•ҝжңҹдёҠдёӢж–Үз»ҙжҠӨвҖҢгҖӮдҫӢеҰӮпјҢдёҖж¬ЎеҜ№иҜқдёӯеҸҜеҗҢж—¶и°ғз”Ёж•°жҚ®еә“гҖҒж–Үд»¶зі»з»ҹе’ҢAPIвҖҢгҖӮ
  • вҖҢFunction CallingвҖҢпјҡеҹәдәҺвҖҢиҜ·жұӮ-е“Қеә”жЁЎејҸвҖҢпјҢжЁЎеһӢи§ҰеҸ‘еҚ•ж¬ЎеҮҪж•°и°ғз”ЁеҗҺзӣҙжҺҘиҝ”еӣһз»“жһңпјҢж— жҢҒз»ӯдәӨдә’иғҪеҠӣвҖҢгҖӮ

ж ёеҝғеҢәеҲ«вҖҢ

з»ҙеәҰ MCPеҚҸи®® Function Calling

вҖҢйҖӮз”ЁеңәжҷҜвҖҢ и·Ёе№іеҸ°йӣҶжҲҗгҖҒеӨҡе·Ҙе…·еҚҸеҗҢгҖҒй«ҳе®үе…ЁжҖ§йңҖжұӮвҖҢ еҚ•жЁЎеһӢгҖҒиҪ»йҮҸзә§еҠҹиғҪеҝ«йҖҹејҖеҸ‘вҖҢ
вҖҢз”ҹжҖҒе…је®№жҖ§вҖҢ ејҖжәҗеҚҸи®®пјҢж”ҜжҢҒеӨҡеҺӮе•Ҷе·Ҙе…·жҺҘе…ҘвҖҢ дҫқиө–зү№е®ҡжЁЎеһӢпјҲеҰӮGPTзі»еҲ—пјүвҖҢ

вҖҢдә’иЎҘе…ізі»вҖҢ

дёӨиҖ…е№¶йқһжӣҝд»Је…ізі»пјҢиҖҢжҳҜдә’иЎҘпјҡ

  • вҖҢMCPвҖҢ и§ЈеҶіеә•еұӮеҚҸи®®ж ҮеҮҶеҢ–й—®йўҳпјҢжҸҗеҚҮе·Ҙе…·и°ғз”Ёзҡ„зҒөжҙ»жҖ§е’ҢеӨҚз”ЁжҖ§вҖҢгҖӮ
  • вҖҢFunction CallingвҖҢ дҪңдёәй«ҳеұӮеҠҹиғҪпјҢдёәзү№е®ҡжЁЎеһӢжҸҗдҫӣеҚіжҸ’еҚіз”Ёзҡ„иғҪеҠӣвҖҢгҖӮ
